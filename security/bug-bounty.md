# Bug bounty program

idOS maintains an active bug bounty program on HackenProof.

## Program details

- **Platform**: [HackenProof — idOS Apps and SC](https://hackenproof.com/programs/idos-apps-and-sc-1)
- **Scope**: idOS applications and smart contracts

## Reporting

If you discover a potential vulnerability, please report it through the HackenProof platform. For critical issues requiring immediate attention, contact [engineering@idos.network](mailto:engineering@idos.network).
